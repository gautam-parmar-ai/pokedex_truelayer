# Pokedex - React Application

A beautiful, interactive Pokedex application built with React and Vite. Browse through 151 Pokemon, search, sort, view details, and discover random Pokemon!

## Features

- Browse 151 Pokemon with beautiful card-based UI
- Search Pokemon by name
- Sort Pokemon by number (ascending/descending)
- Grid and List view modes
- Click on any Pokemon to view detailed information
- "Surprise Me!" button for random Pokemon discovery
- Detailed Pokemon page with stats, abilities, and description
- Fully responsive design
- Clean, modern UI with smooth animations

## Tech Stack

- **React 19** - UI Framework
- **Vite 7** - Build tool and dev server
- **React Router DOM 7** - Routing and navigation
- **PokeAPI** - Pokemon data source
- **Manrope Font** - Modern typography
- **CSS3** - Styling with flexbox, grid, and animations

## Getting Started

### Prerequisites

Before you begin, ensure you have installed:
- **Node.js** (version 18 or higher)
- **npm** (comes with Node.js) or **yarn**

### Installation

1. **Clone the repository**
   ```bash
   git clone <your-repo-url>
   cd pokedex
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start the development server**
   ```bash
   npm run dev
   ```

4. **Open your browser**
   
   Navigate to: `http://localhost:5173`
   
   (If port 5173 is busy, Vite will use another port - check the terminal output)

### Build for Production

To create a production build:

```bash
npm run build
```

The built files will be in the `dist` folder.

To preview the production build locally:

```bash
npm run preview
```

## API Reference

This project uses the [PokeAPI](https://pokeapi.co/):

- **Pokemon Data**: `https://pokeapi.co/api/v2/pokemon/{id}`
- **Pokemon Species**: `https://pokeapi.co/api/v2/pokemon-species/{id}`
- **Pokemon List**: `https://pokeapi.co/api/v2/pokemon?limit=151`

## Customization

### Changing the Number of Pokemon

Edit `src/Pokemon.jsx`:
```javascript
const API = "https://pokeapi.co/api/v2/pokemon?limit=151";
// Change 151 to any number (max 10000 for PokeAPI)
```

### Adding More Pokemon Types

Edit the `typeColors` object in:
- `src/PokemonCards.jsx`
- `src/PokemonDetail.jsx`

```javascript
const typeColors = {
    // Add new types here
    newType: "#HEXCODE",
};
```

