# Pokedex TrueLayer

A Pokedex application built with React and Vite.

## Live Demo

Access the deployed application at: https://gautam-parmar-ai.github.io/pokedex_truelayer

## Getting Started

1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```
4. Build for production:
   ```bash
   npm run build
   ```

## Tech Stack

- React
- Vite
- React Router
